# Campus Connect + AI Assistant

A MERN stack project for campus social networking with AI assistant features (e.g., chat, recommendations).

## Structure
- `/backend`: Node/Express server, MongoDB models/routes.
- `/frontend`: React app for UI.

## Quick Start
1. Backend: `cd backend && npm install && npm run dev` (set MONGODB_URI in .env)
2. Frontend: `cd frontend && npm install && npm start`
3. Open http://localhost:3000

## Features (Planned)
- User auth (login/register)
- Posts/feed
- AI chat assistant

## Tech
- Frontend: React, Axios
- Backend: Express, Mongoose, JWT
- DB: MongoDB
